
## 2.4.0 - 2026-04-22

### Changed

- Allows to use `symfony/serializer` with PHP 8.1 compatible versions (6.4.x)

For development purpose:

- Applied migration of CaptainHook config files to use the
[`bartlett/captainhook-bin-plugin v1.1`](https://github.com/llaville/captainhook-bin-plugin/releases/tag/1.1.0) plugin
- Added [ECS](https://github.com/easy-coding-standard/easy-coding-standard) usage on Git Hooks via CaptainHook
- Added [Codeception](https://github.com/Codeception/Codeception) usage on Git Hooks with CaptainHook
- PHPUnit pre-push hook on CaptainHook was disabled (replaced by Codeception)

### Removed

- Unused `php-parallel-lint/php-console-highlighter` dependency since v2.0.0,
used by converters that are now part of [`bartlett/sarif-php-converters`](https://github.com/llaville/sarif-php-converters) project

**Full Changelog**: [2.3.0...2.4.0](https://github.com/llaville/sarif-php-sdk/compare/2.3.0...2.4.0)
